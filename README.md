- **实战导向**：每个例程都配备说明文档，帮助理解应用场景及编程逻辑，快速上手实际项目开发。

1. **克隆仓库**：点击仓库右上角的"Clone or download"按钮，选择合适的下载方式获取全部资源。

3. **导入至编程软件**：使用TIA Portal（ Totally Integrated Automation Portal）或其他兼容的西门子编程工具导入例程。
